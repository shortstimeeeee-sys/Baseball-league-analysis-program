경기 기록 가공

경기 목록

단계형으로 경기 완성: 1) 날짜/시간 확정 → 2) 중계 텍스트 파싱 → 3) 경기 저장(진행중) → 4) 기록 확인 후 완료 처리.

이닝 인식 방법
텍스트에 1회초 LG 공격, 1회말 한화 공격, 2회초 LG 공격처럼 「N회초/말 + 팀명 + 공격」 줄이 나올 때마다, 그 다음에 오는 타석들이 해당 이닝으로 들어갑니다.
1회초 이후 기록: 1회말·2회초·2회말… 도 넣으려면, 위 형식의 줄을 각 이닝마다 포함한 전체 중계문을 한 번에 붙여넣어 주세요. 가져오기할 때마다 기존 기록이 새 내용으로 통째로 바뀌므로, 1회초만 넣고 나중에 1회말만 추가하는 식으로는 이어 붙이기가 되지 않습니다.

1단계 · 경기 날짜/시간 (파싱 전에 선택)

일정 · 더블헤더

아래에서 경기 날짜·시간을 정한 뒤, 시범 경기 여부와 그날 같은 상대의 2차전(더블헤더) 여부를 체크하세요.

체크하지 않으면 같은 날·같은 매치업(같은 리그) 경기가 이미 있으면 기존 경기에 기록을 합칩니다.
